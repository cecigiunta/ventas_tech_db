# Ventas Tech DB

## Descripción
Este proyecto corresponde al trabajo práctico integrados del Módulo 3 del Curso Data Analytics.

## Propósito del ejercicio
En esta práctica se construye desde cero la base de datos Ventas_Tech_DB — el dataset que utilizaré durante todo el curso.
Se define la estructura de cuatro tablas relacionadas, las claves foráneas y la carga de los primeros datos para consultar.

## Tecnologías utilizadas
- SQL
- Git
- GitHub


## Funcionalidades
- Creación de la base de datos
- Creación de las tablas: clientes, productos, ventas y categorías
- Inserción de registros
